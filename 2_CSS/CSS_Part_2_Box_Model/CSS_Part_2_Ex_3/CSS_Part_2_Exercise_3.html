1ITF

In the first phase of our IT Factory you’ll become acquainted with the IT world and acquire a thorough knowledge of the different IT domains. You’ll learn to develop applications in Python and Java and become an expert in web design. You’ll analyse data with SQL and develop a solid foundation in computer networks and operating systems.

You’ll learn by doing and immediately apply the newly acquired knowledge and skills in various projects. This way, you won’t only acquire technical expertise but also communication skills and soft skills.

2ITF

In the second phase you’ll gain insight into the relationship between IT and business, learning to translate your client’s wants and needs into concrete models.

In addition, you’ll practice object-oriented analysis and acquire in-depth knowledge of PHP, Java, WPF, ASP.Net, MVC and Cordova. You’ll learn to replace existing, often inefficient business processes with new technologies in order to create services that meet your customer's needs.

3ITF

In the third phase, you’ll become an expert in the fascinating world of state-of-the-art technology. Capture, manage, visualise and analyse object-oriented ‘big data’. This technology will allow you to replace existing, often inefficient, business processes with new technologies. With this goal in mind, you’ll be immersed in the wonderful world of Aritficial Intelligence, Big Data and Business Intelligence.
